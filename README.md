### 1. Interactive Traction Control System (TCS) Simulator

Under my academic projects section, users can launch an embedded canvas-based 2D physics simulation of an active traction controller.

- **Dynamic Slip Ratio Plotting:** View a real-time running telemetry graph tracking actual vs. target wheel slip.
- **Surface Coefficients:** Test dynamic PID responses on variable friction surfaces (Ice, Wet Floor, Dry Wood).
- **Toggleable Traction Control (TCS):** Toggle the feedback loop on/off to see wheel slip drop by up to 63% on high-load surfaces.
